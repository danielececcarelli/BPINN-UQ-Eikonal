Compute alpha=acceptance rate.

We modify the formula in order to prevent getting stuck in local minimum. First compute h = H(U,r)-H(U0,r0). If this is positive -> we want to reject, so we return an high alpha (max 0.95 and not 1.0 for the previous reason) If this is negative we'll return a small alpha, using the formula alpha = exp(2*h), with h negative
