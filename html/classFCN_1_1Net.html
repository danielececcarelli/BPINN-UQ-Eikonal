Detailed Description

Define the base neural network model: Fully Connected Network.

Model defined using tf.keras.Sequential() The final architecture will be: [n_input, n_hidden_neuron, n_hidden_neuron,...,n_hidden_neuron, n_output] with n_hidden_neuron repated for n_hidden_layer times

Parameters
n_inputinput dimension (1, 2 or 3)
n_hidden_layernumber of hidden layers
n_hidden_neuronnumber of neurons for each hidden layer
n_outputdimension of output = n_output_vel+1 (1 node for activation times, n_output_vel for conduction velocity)
